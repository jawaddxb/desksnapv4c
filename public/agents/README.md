# Agent Avatars

This directory contains comic book-style avatar images for the AI agent team.

## Required Images

| File | Agent | Color | Description |
|------|-------|-------|-------------|
| `scout.png` | Scout | #4A90A4 (teal-blue) | Explorer/detective with magnifying glass |
| `sage.png` | Sage | #6B8E6B (brand green) | Architect with blueprints/grid overlay |
| `aria.png` | Aria | #C9A962 (warm gold) | Creative writer with flowing pen |
| `nova.png` | Nova | #A78BFA (purple) | Artist with palette/paint splashes |
| `coach.png` | Coach | #D4854A (warm orange) | Wise mentor with lightbulb halo |

## Image Specifications

- **Format**: PNG with transparency
- **Size**: 200x200px (will be displayed at 40x40px to 48x48px)
- **Style**: Comic book / illustrated character style
- **Background**: Transparent

## Generation Prompt

Use an AI image generator with a prompt like:

```
Comic book style character portrait, [description], vibrant colors,
clean lines, professional illustration, white/transparent background,
centered face/torso view, 200x200px
```

For example, for Scout:
```
Comic book style character portrait, explorer detective with magnifying glass,
teal blue color scheme (#4A90A4), vibrant colors, clean lines,
professional illustration, transparent background, centered face/torso view
```
